-------------------------------------------------------

-------------------------------------------------------
git commit -m “XXX”    输入修改备注
-------------------------------------------------------
git status               查看仓库当前状态
-------------------------------------------------------
git diff XXX.txt         查看修改内容
-------------------------------------------------------
git log                  查看历史版本
-------------------------------------------------------

-------------------------------------------------------
cat XXX.txt              查看当前版本文件内容
-------------------------------------------------------

-------------------------------------------------------
git reflog               查看每一次命令 
-------------------------------------------------------

-------------------------------------------------------

-------------------------------------------------------
git checkout -- XXX      丢弃工作区修改
-------------------------------------------------------
git rm XXX.txt           删除版本库中文件
-------------------------------------------------------
git checkout -- XXX      恢复版本库中文件
-------------------------------------------------------
git checkout -- XXX.txt  将文件在工作区的修改全部撤销
